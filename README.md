## How To Use

run install depedencies
```sh
npm install 
```

runing migration 
```sh
npx sequelize-cli db:migrate
```

config database in folder `config/config.json`



run node app
```sh
node ./bin/www
```

server runing in port 6500 ----> cek in folder bin/www
